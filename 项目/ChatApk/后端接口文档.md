# 后端接口文档

地址:`http://localhost:3000`

## 注册

- 地址: `/signup/add`

- 请求方式: POST

- 字段及类型说明

  |   字段   |  类型  |  说明  | 必须 |
  | :------: | :----: | :----: | :--: |
  |   name   | String | 用户名 |  是  |
  | password | String |  密码  |  是  |
  |  email   | String |  邮箱  |  是  |

- 输出

  |    字段    |  类型  |       说明        | 必须 |
  | :--------: | :----: | :---------------: | :--: |
  | statusCode | String | 状态码(200为成功) |  是  |

## 邮箱或者用户是否存在验证

- 地址: `/signup/judge`

- 请求方式: POST

- 字段以及类型说明

  | 字段 |  类型  |    说明     | 必须 |
  | :--: | :----: | :---------: | :--: |
  | data | String | 邮箱/用户名 |  是  |
  | type | String | name/email  |  是  |

- 输出

  |    字段    |  类型  |             说明             | 必须 |
  | :--------: | :----: | :--------------------------: | :--: |
  | statusCode | String |     状态码(200表示成功)      |  是  |
  |   result   | Number | 在数据库中查找到的数据的条数 |  是  |

## 判断搜索的用户是否是好友

- 地址: `/search/isfriend`

- 请求方式： POST

- 字段以及类型说明

  | 字段  |  类型  |         说明          | 必须 |

  |  uid  | String |        用户ID         |  是  |
  |  fid  | String |        好友ID         |  是  |

  | token | String |         验证          |  是  |

- 输出

  |   字段    |  类型  |            说明             | 必须 |
  | :-------: | :----: | :-------------------------: | :--: |
  | stateCode | String | 状态码(200表示已经是好友了) |  是  |

## 用户信息更新

- 地址: `/user/update`

- 请求方式: POST

- 字段以及类型说明

  - 邮箱以及密码修改

    |    字段     |  类型  |             说明              | 必须 |
    | :---------: | :----: | :---------------------------: | :--: |
    |     id      | String |            用户ID             |  是  |
    |    data     | String |         新密码/新邮箱         |  是  |
    |    type     | String |        password/email         |  是  |
    | oldPassword | String | 修改密码/邮箱都需要传入旧密码 |  是  |
    |    token    | String |             验证              |  是  |

  - 其余项修改

    | 字段 |  类型  |      说明       | 必须 |
    | :--: | :----: | :-------------: | :--: |
    |  id  | String |     用户ID      |  是  |
    | data | String |   修改后的值    |  是  |
    | type | String | sex/imgUrl/name |  是  |

- 输出

  |    字段    |  类型  |          说明           | 必须 |
  | :--------: | :----: | :---------------------: | :--: |
  | statusCode | String | 状态码(200表示修改成功) |  是  |
  |            |        |                         |      |
