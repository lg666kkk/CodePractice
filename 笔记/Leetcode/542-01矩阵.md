# 542.01矩阵

## 题目描述

给定一个由 0 和 1 组成的矩阵，找出每个元素到最近的 0 的距离。

两个相邻元素间的距离为 1 

## 示例

### 示例1

```java
输入：
0 0 0
0 1 0
0 0 0
```

```javascript
输出：
0 0 0
0 1 0
0 0 0
```

### 示例2

```javascript
输入：
0 0 0
0 1 0
1 1 1
```

```javascript
输出：
0 0 0
0 1 0
1 2 1
```

**注意：**

1. 定矩阵的元素个数不超过 10000。
2. 给定矩阵中至少有一个元素是 0。
3. 矩阵中的元素只在四个方向上相邻: 上、下、左、右。
