3. 对数据库的高可扩展性和高可用性的需求

#### 创建数据库
1. show dbs 命令可用于查看有哪些数据库
2. 创建数据库
   - use 数据库名
   - db.user.insert({"eee":"www"})  -- 插入一条数据后即可成功创建数据库
3. 查看数据库有哪些表： show collections
4. 创建表 
   - use 数据库名  -- 进入数据库
   - db.表名.insert({"eee":"www"})  -- 向表中插入一条数据即可成功创建表
